# Final Report Package

This directory contains the report deliverables and the source files needed to rebuild or audit the PDF.

- `ARIN_Group_Report.pdf`: final report PDF.
- `ARIN_Group_Report.md`: report source.
- `assets/`: figures, screenshots, and editable SVG sources used by the report.
- `evidence/`: copied evaluation evidence referenced by the appendix and Figure 9.
- `templates/`: archived HKU dissertation template used as a formatting reference.
- `references.bib`, `ieee.csl`, `report-style.tex`: citation and PDF styling inputs.
